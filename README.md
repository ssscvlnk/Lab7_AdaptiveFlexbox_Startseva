## Описание
**Цель работы:** освоить современные технологии CSS-разметки — Flexbox.
**Адаптивная верстка** — это подход к созданию веб-страниц, при котором дизайн
и контент автоматически подстраиваются под размер экрана устройства (компьютер,планшет, смартфон).
**Flexbox** — одномерная система компоновки для размещения элементов в строку или столбец.
## Теория. Введение в Flexbox
**Flexbox (Flexible Box Layout)** — это модуль CSS для создания гибких одномерных
раскладок. Он идеально подходит для:
- Выравнивания элементов по горизонтали или вертикали
- Распределения свободного пространства
- Создания навигационных меню
- Центрирования элементов
Основные концепции
**Flex-контейнер** — родительский элемент с display: flex
**Flex-элементы** — дочерние элементы внутри контейнера
```
<div class="container">
    <div class="item">1</div>
    <div class="item">2</div>
    <div class="item">3</div>
</div>
```
## Свойства Flex-контейнера
### 1. Основные свойства
- display: flex — создает flex-контейнер
- flex-direction — определяет направление главной оси:
    - row — слева направо (по умолчанию)
    - row-reverse — справа налево
    - column — сверху вниз
    - column-reverse — снизу вверх
### 2. Выравнивание элементов
- justify-content — выравнивание по главной оси:
    - flex-start — по началу
    - flex-end — по концу
    - center — по центру
    - space-between — равномерно с отступами
    - space-around — равномерно с отступами по краям
    - space-evenly — равномерно с равными отступами
- align-items — выравнивание по поперечной оси:
    - stretch — растянуть (по умолчанию)
    - flex-start — по началу
    - flex-end — по концу
    - center — по центру
    - baseline — по базовой линии
### 3. Перенос элементов
- flex-wrap — управление переносом:
    - nowrap — без переноса (по умолчанию)
    - wrap — перенос на новую строку
    - wrap-reverse — обратный перенос
### 4. Отступы между элементами
- gap — устанавливает отступы между элементами
```
.container {
    display: flex;
    flex-direction: row;
    align-items: baseline;
    flex-wrap: wrap;
    gap: 10px;
}
```
## Свойства Flex-элементов
### 1. Гибкость элементов
- flex-grow — определяет, насколько элемент может увеличиваться
- flex-shrink — определяет, насколько элемент может сжиматься
- flex-basis — базовый размер элемента
- flex — шорткат для flex-grow, flex-shrink и flex-basis
### 2. Индивидуальное выравнивание
- align-self — переопределяет выравнивание для отдельного элемента
```
.item {
    flex-basis: 150px;
    flex-shrink: 0;
    flex: 1;
    align-self: flex-end;
}
```
